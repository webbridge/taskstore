# TaskStore

For starting project you need to

## Run database via docker-compose

For installing docker compose please visit https://docs.docker.com/compose/install/

```
docker-compose up
```

## Run server side :

```
CD ./server
npm install (for installing all packages)
npm start
```

## Run server side :

```
CD ./frontend
npm install (for installing all packages)
npm start
```
